App for storing and rating media like movies, series, books and games or to store favourite recipes, restaurant or travel destinations. Create your custom category or subcategory, create custom rating fields and store ratings in a list which you can later sort by any of your created fields or tags. Add rating fields that use numerical 1-100 scale or a text based comment.

Example use case:
1.
Create a category "Movies", subcategory "Horror". Create rating fields like "plot", "acting", "production", "overall", add a "director" or "production house" field and add some custom tags like "horror", "slow-burn", "dark".

Create a category "Food", subcategory "desserts". Create rating fields like "Rating", "Cost" or "Pairs good with...". Add some custom tags like "Sweet", "Chocolate", "Easy-to-make".
